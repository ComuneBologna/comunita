-- SUMMARY --

As an extension to Multiupload Filefield Widget 
(http://drupal.org/sandbox/czigor/1115362) this module presents the same 
widget to Image fields.

-- REQUIREMENTS --

The core Image module.

-- INSTRUCTIONS --

1. Enable the module.
2. Add an 'image' field to a content type and select the widget 'Multiple'.
3. Done

-- BROWSERS --

Known browsers to work with: Firefox 3.6, Chromium 10, Opera 11.01.
As no version of IE supports the 'multiple' html5 attribute, I'm pretty sure 
that it does not work with IE.
