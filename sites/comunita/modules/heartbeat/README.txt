-- OUT OF THE BOX --

Heartbeat module alone is the API module. You could log and display activity in a custom module. However
it can work out of the box by enabling the heartbeat_defaults and heartbeat_rules module. Rules will log
activity by using the default heartbeat templates, shown in the heartbeat streams.
